# Real-Time Engineering Roadmap Simulator

This project is built for the **AI Partner Catalyst Hackathon**.

## Tech Stack
- Confluent Cloud (Kafka)
- Machine Learning (Scikit-learn)
- Streamlit Dashboard
- Google Cloud (Deployment)

## How it Works
1. Student progress is streamed in real-time using Kafka.
2. ML model predicts the next learning step.
3. Live dashboard shows roadmap updates instantly.

## How to Run
1. Start Kafka producer
2. Start roadmap consumer
3. Run Streamlit dashboard
